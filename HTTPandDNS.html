This page contains information on Hypertext Transfer Protocall and the Domain Name System




The Hypertext Protocall


The Hypertext Transfer Prototcall is the world wide web. It uses a lot of GET requests to obtain packets from other sources to display images, videos, and other forms of media.

DNS


DNS (Domain Name System) is a way of translating IP adresses into something understandble by humans. Instead of having a whole bunch of numbers the DNS stores the IP adress as a .com site. The DNS is a system of serevers that contain the infromation.
